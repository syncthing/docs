DeviceDiscovered
----------------

Emitted when a new device is discovered using local discovery.

.. code-block:: json

    {
        "id": 13,
        "globalID": 13,
        "type": "DeviceDiscovered",
        "time": "2014-07-17T13:28:05.043465207+02:00",
        "data": {
            "addrs": [
                "172.16.32.25:22000"
            ],
            "device": "NFGKEKE-7Z6RTH7-I3PRZXS-DEJF3UJ-FRWJBFO-VBBTDND-4SGNGVZ-QUQHJAG"
        }
    }
